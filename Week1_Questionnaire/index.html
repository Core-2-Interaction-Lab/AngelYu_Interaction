  6. Describe the relationship between HTML, CSS and Javascript.

    7. HTML and CSS are front-end coding--what the user can visually see. Html is the basic structure of a website, which contains words, symbols, numbers, and boxes. Html is very basic, simple, and will not look pretty. An analogy for this is the barebones of a building: the pillars, windows, doors, and frames. In addition, adding rooms, like a bathroom, living room, offices, and overall separating the space into different rooms.

    Css is the tool to decorate your HTML, making your basic default font and size look pretty. CSS lets you also change the color of any element, adjust how the window interacts with different resolution sizes, add animation, shapes, and edit or add an image. An analogy for this could be adding plants, wallpaper, lights, and flooring for a building.

    Javascript is back-end coding of a website. The element you can not visually see when you first look at a website, but until you inspect the page.

    Javascript affects how the user interacts with the HTML elements and how a website functions. Adding sounds, pushing buttons, and advanced animations when hovering, clicking, or scrollig through a website.

    An anaology of this is the electrical, plumbing, cooling, and heating system of a building or the hinges of the doors and windows.
